## Default Permission

This permission set configures which scanleaf features are exposed by default.

#### Granted Permissions

Grants access to all camera lifecycle and scanning functionality.

#### This default permission set includes the following:

- `allow-open-camera`
- `allow-close-camera`
- `allow-start-scan`
- `allow-stop-scan`
- `allow-check-permissions`
- `allow-request-permissions`
- `allow-open-app-settings`
